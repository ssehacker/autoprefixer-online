### Run server on local
You need install node(version 4+) , npm(version 3+) and webpack(version 1.x) fist, then
Clone project:
```
$ git clone https://github.com/ssehacker/autoprefixer-online.git
```

Install dependencies:
```
$ npm install
```

Compile:
```
$ npm run build
```

Run server:
```
$ npm start
```

open browser and visit `localhost:3004`.
